# EMSD Raw Data

Public EMSD Hong Kong Energy End-use Data files used for calibration.

## Files

- `energy_end_use_table12.csv`: end-use breakdown shares, especially air-conditioning and commercial load-shape assumptions.
- `electricity_consumption_by_sector_table08.csv`: territory-wide electricity consumption by residential, commercial, industrial, and transport sectors.

Public dataset page:

https://data.gov.hk/en-data/dataset/hk-emsd-emsd1-energy-end-use-data-2025
