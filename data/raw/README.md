# Raw Public Data

This folder contains curated public source datasets used to calibrate the Hong Kong grid model. Generated solver artifacts, local databases, and processed exports remain ignored by git.

## Current Files

### `hk_electric/consumption_by_district_and_customer_type.csv`

- Source: HK Electric open data / data.gov.hk
- Dataset: Consumption Data by Districts and Customer Type
- Public source page: https://data.gov.hk/tc-data/dataset/hkelectric-cs_cs-consumption-data-by-districts-and-customer-type
- Use in model: allocate Hong Kong Island and Lamma demand by district and sector.

### `hk_electric/consumption_by_customer_type.csv`

- Source: HK Electric open data / data.gov.hk
- Dataset: Consumption Data by Customer Type
- Public source page: https://data.gov.hk/en-data/dataset/hkelectric-cs_cs-consumption-data-by-customer-type
- Use in model: calibrate residential, commercial, and industrial sector totals for HK Electric territory.

### `emsd/energy_end_use_table12.csv`

- Source: EMSD Hong Kong Energy End-use Data / data.gov.hk
- Dataset table: end-use breakdown table downloaded as `table12.csv`
- Public source page: https://data.gov.hk/en-data/dataset/hk-emsd-emsd1-energy-end-use-data-2025
- Use in model: sector/end-use calibration, especially air-conditioning and commercial load-shape assumptions.

## Expected Additional Sources

### `clp/`

Place CLP public consumption, network, or open-data exports here. CLP data is needed for Kowloon, New Territories, and many outlying islands.

### `census_statistics/`

Place Census and Statistics Department Hong Kong Energy Statistics tables here. These are useful for territory-level monthly/annual validation.

### `osm/`

Place reproducible OSM/Overpass exports here only if they are small enough to review. Large extracts should stay outside git and be regenerated by scripts.

### `data_centers/`

Place manually curated public data-center site/capacity evidence here, preferably as CSV with source URL columns.

## Git Policy

Only raw public datasets and small provenance files should be committed from this folder. Do not commit:

- `data/*.sqlite3`
- generated PowerModels/GridSFM JSON exports
- solver scenario outputs
- large Overpass extracts that can be regenerated
